# KeymanMCP — 1-Page Internship Opportunity

## The Primary Project

You'll build **KeymanMCP**, a Python MCP (Model Context Protocol) server that teaches AI assistants how to create and adapt Keyman keyboards. **Keyman** ([keyman.com](https://keyman.com)) is the world's largest keyboard engine for lesser-resourced languages—it powers keyboards for over 1,000 languages used by linguists, missionaries, and indigenous communities. Right now, creating a keyboard requires semi-manually coordinating 6+ interdependent files (.kmn, .kps, .kvks, .keyman-touch-layout, documentation)—a process that's error-prone and inaccessible to non-technical users.

**KeymanMCP changes that.** AI assistants will scaffold complete keyboards from scratch in <5 seconds, safely adapt existing keyboards in <2 seconds, and validate consistency across all files automatically. You'll implement 5 MCP tools (generate_keyboard, build_keyboard, validate_consistency, copy_keyboard, inspect_keyboard), build a pre-built API index of Keyman's language specification (256+ virtual key names, 30+ system stores, 120+ error codes, and code patterns), and integrate with the `kmc` CLI as a subprocess.

## What You'll Learn

- **Python**: Async programming, file I/O, data structures, subprocess management, testing
- **API design**: How to document systems so machines (and humans) understand them
- **File format parsing**: XML, JSON, regex, validation across multiple interdependent formats
- **Collaborative development**: Pair-programming, code review, daily standups, shared ownership
- **Real-world shipping**: Building production code that people actually use

You'll learn patterns that apply far beyond Keyman—how to extract complex domain knowledge and make it accessible to AI. This is a concrete example of teaching AI to code correctly.
